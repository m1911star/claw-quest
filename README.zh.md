# ClawQuest 中文指南

> **把每一次 AI 对话，变成角色成长**
> 每次聊天自动获得经验，刷装备，升等级，完成任务——无需额外操作。

ClawQuest 是一个 OpenClaw Skill，为你的 AI 助手加上完整 RPG 系统。正常使用 AI 就能自动涨经验，对话越多角色越强。

---

## 🎮 它能做什么？

- 六种**职业**（学者 / 工匠 / 斥候 / 使者 / 贤者 / 守护者）
- 六项**属性**（INT 智识 / CHA 口才 / WIS 洞察 / DEX 创造 / CON 效率 / STR 毅力）
- **100 级**成长曲线 + 指数型经验需求
- **装备系统**：6 品质 × 随机词缀，从白装到神话
- **三种任务**：修炼（习惯）/ 日常委托 / 冒险任务
- **金币经济**：任务奖励 + 商店购买
- **连击加成**：连续使用天数越多，经验倍率越高

> 纯粹游戏层——不影响 AI 真实能力，只是让交互更有趣 😄

---

## 🚀 快速开始

### 安装

```bash
# 推荐：通过 ClawHub 安装
clawhub install clawquest

# 手动安装
cd ~/.openclaw/skills/
git clone https://github.com/m1911star/claw-quest.git clawquest
```

**安装完成后，运行一次注册脚本：**

```bash
python3 ~/.openclaw/skills/clawquest/scripts/install.py
```

这会自动将全局经验规则写入你的 `AGENTS.md`——之后无论在哪个群组、私聊还是频道，只要触发 OpenClaw 就会自动发放经验 ✨

如需卸载：

```bash
python3 ~/.openclaw/skills/clawquest/scripts/install.py --uninstall
```

### 创建角色

```
/cq new
```

AI 会引导你完成「觉醒仪式」——几个简单问题后，根据你的使用习惯推荐职业。

### 开始冒险

正常聊天就行！每轮对话自动涨经验，用 `/cq` 查看完整面板。

## 📖 核心系统

### 六大职业

| 职业 | 主属性 | 适合场景 |
|------|--------|----------|
| 📚 **学者 Scholar** | INT | 知识问答、研究、学习 |
| 🔨 **工匠 Artisan** | DEX | 编程、设计、内容创作 |
| 🔍 **斥候 Scout** | WIS | 搜索、数据分析、调查 |
| 📜 **使者 Herald** | CHA | 翻译、写作、社交媒体 |
| 🔮 **贤者 Sage** | WIS | 规划、策略、决策支持 |
| 🛡️ **守护者 Guardian** | CON | 自动化、批处理、工作流 |

**职业加成**：使用主属性相关功能时 +20% 经验

### 经验奖励标准

| 交互类型 | XP |
|----------|----|
| 简单问候、单句回答 | 1-3 |
| 普通帮助请求 | 5-10 |
| 复杂任务、编码、深度解释 | 15-30 |
| 长对话、研究、多轮规划 | 20-50 |
| 完成冒险任务 | 30-80 |

### 连击加成

| 连续天数 | 倍率 |
|----------|------|
| 3 天 | ×1.2 |
| 7 天 | ×1.5 |
| 14 天 | ×1.8 |
| 30 天 | ×2.0 |

## 🎯 命令速查

| 命令 | 说明 |
|------|------|
| `/cq` | 查看完整角色面板 |
| `/cq status` | 一行状态栏 |
| `/cq quest` | 查看任务日志 |
| `/cq inventory` | 查看背包 / 装备 |
| `/cq equip <物品>` | 装备物品 |
| `/cq shop` | 打开金币商店 |
| `/cq allocate <属性> <点数>` | 分配属性点 |
| `/cq adventure add <名称> <XP> <品质>` | 添加冒险任务 |
| `/cq adventure done <名称>` | 完成冒险，触发掉宝 |
| `/cq daily add <名称> <XP> <周期>` | 添加日常委托 |
| `/cq daily done <名称>` | 完成日常 |
| `/cq discipline add <名称> <XP>` | 添加修炼 |
| `/cq discipline + <名称>` | 记录一次正向修炼 |
| `/cq quiet` | 开/关静默模式（隐藏状态栏） |
| `/cq help` | 帮助列表 |

## 🛠️ 技术说明

- **纯 Python 3**，无第三方依赖
- 所有数值计算在脚本里完成，不依赖 LLM 算数
- 状态文件：`~/.openclaw/skills/clawquest/data/character.json`
- `install.py` 注入 AGENTS.md 的方式是带 marker 的 idempotent patch，支持 `--uninstall` 干净卸载
