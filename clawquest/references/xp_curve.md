# ClawQuest XP Curve & Level Progression

This document defines the experience point requirements for each level from 1 to 100.

---

## 📈 XP Curve Formula

```
XP_required(level) = floor(50 × 1.08^(level - 1))
```

**Explanation:**
- Base XP: 50 (level 1→2)
- Growth rate: 8% per level (exponential)
- Levels 1-100 total: ~500,000 XP

## 📅 Estimated Time to Level 100

**Assumptions:**
- Average 10 XP per conversation turn
- 20 conversations per day
- No streaks or class bonuses
- Total XP needed: ~500,000

**Calculation:**
- XP per day: 10 × 20 = 200
- Days to level 100: 500,000 / 200 = 2,500 days (~7 years)

**With bonuses (class + 30-day streak):**
- Effective XP per turn: 10 × 1.2 × 2.0 = 24
- XP per day: 24 × 20 = 480
- Days to level 100: 500,000 / 480 = ~1,042 days (~3 years)

**Realistic estimate for active users:** 1-3 years to level 100

## 🧮 Python Implementation Reference

### XP Calculation Function

```python
import math

def xp_for_level(level: int) -> int:
    """Calculate XP required to reach next level."""
    return math.floor(50 * (1.08 ** (level - 1)))

def cumulative_xp_for_level(level: int) -> int:
    """Calculate total XP required to reach a specific level."""
    return sum(xp_for_level(i) for i in range(1, level))

def calculate_level(current_xp: int) -> int:
    """Determine current level from total XP."""
    level = 1
    while cumulative_xp_for_level(level + 1) <= current_xp:
        level += 1
    return level

def xp_to_next_level(current_xp: int, current_level: int) -> int:
    """Calculate remaining XP needed for next level."""
    next_level_xp = cumulative_xp_for_level(current_level + 1)
    return next_level_xp - current_xp
```

### Level-Up Detection

```python
def check_level_up(old_xp: int, new_xp: int) -> dict:
    """Check if adding XP causes a level-up."""
    old_level = calculate_level(old_xp)
    new_level = calculate_level(new_xp)
    
    return {
        "leveled_up": new_level > old_level,
        "old_level": old_level,
        "new_level": new_level,
        "levels_gained": new_level - old_level,
        "skill_points_gained": (new_level - old_level) * 2
    }
```

## 🎮 Tier-Specific Unlocks

### Apprentice (1-10)
- Basic equipment drops (white, green)
- All core features available
- Learning the system

### Traveler (11-25)
- Rare (blue) equipment drops unlocked
- Shop discounts (10% off)
- First major milestone rewards

### Master (26-50)
- Epic (purple) equipment drops unlocked
- Class change discount (200 → 150 gold)
- Attribute respec discount (50% off)
- Prestige recognition

### Legend (51-99)
- Legendary (orange) equipment drops unlocked
- Mythic (red) equipment drops (1% on orange quests at 80+)
- Exclusive titles
- Elite status

### Myth (100)
- Guaranteed mythic equipment
- +5 permanent boost to all stats
- Ultimate achievement
- Max level benefits
